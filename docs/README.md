# Adventure Craft Documentation

Welcome to the Adventure Craft documentation! Adventure Craft is an AI-powered voxel world game built with React, Three.js, and Google Gemini AI.

## Overview

Adventure Craft is a Minecraft-inspired browser-based game featuring:

- **AI-Generated Content** — Structures, characters, and items generated by Google Gemini AI
- **Procedural Terrain** — Dynamic world generation with varied biomes
- **Combat System** — Fight enemies, complete quests, gain XP and level up
- **Gold Economy** — Earn gold from enemies, buy items from merchants
- **NPC Dialogue** — AI-powered conversations with friendly characters
- **Crafting & Inventory** — Gather resources and craft items
- **Tree Cutting** — Use axes to chop trees (multiple hits required)

## Quick Start

```bash
# Install dependencies
npm install

# Set your Gemini API key in .env
echo "GEMINI_API_KEY=your-key-here" > .env

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## New Features

### Gold & Economy

- Kill enemies to earn gold
- Shop at Merchant NPCs to buy weapons, armor, and items

### Combat & Progression

- Gain XP from defeating enemies
- Level up to increase attack, speed, and defense
- Equip shields for doubled defense

### Tree Cutting

- Trees require 5-10 hits to cut down
- Use an Axe for 10 damage per hit (vs 2 for fists)
- Get 2-5 wood logs when tree breaks with axe

### Smart Spawning

- Entities spawn away from the player (5+ blocks)
- Non-aquatic creatures won't spawn in water
- Entity caps prevent overpopulation
